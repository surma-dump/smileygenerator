--- Smileygenerator ---
You dont have much possibilitys to admin this tool (sorry).
When you want to add a Smiley for users, put an
PNG-Image in the "smileygenerator"-folder.
It must have the name: "smiley_<key>.png"
<key> is the name of the smiley which you have to 
type in the BBCode.
